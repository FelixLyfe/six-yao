# AI 六爻占卜 MVP

本仓库为 AI 六爻占卜产品的 MVP 初始化工程，包含前端（Next.js）、后端（Hono）与数据库 schema。

## 目录结构

- frontend: Next.js 前端
- backend: Hono 后端
- database: 数据库 schema 与说明
- docs: 产品与技术文档

## 快速开始

安装依赖：

```bash
npm install
```

后端：

```bash
npm run dev:be
```

前端：

```bash
npm run dev:fe
```
