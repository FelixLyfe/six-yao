AI 六爻占卜工具。

核心能力： 1. 用户输入问题 2. 系统生成六爻卦象 3. AI 根据卦象进行解读 4.
用户获得趋势分析与建议

定位：

传统文化娱乐工具 + AI 解读服务

避免宣称预测未来。

# 四、功能需求

## 1 首页

功能：

-   输入占卜问题
-   开始占卜按钮

字段：

question

------------------------------------------------------------------------

## 2 起卦系统

用户点击投掷6次

每次生成一个爻

状态：

6 老阴 7 少阳 8 少阴 9 老阳

------------------------------------------------------------------------

## 3 卦象计算

规则

阴阳转换：

6 阴 7 阳 8 阴 9 阳

前三爻 = 下卦\
后三爻 = 上卦

组合生成 64 卦

------------------------------------------------------------------------

## 4 变卦计算

规则：

6 → 7\
9 → 8

重新生成卦象

------------------------------------------------------------------------

## 5 AI解卦

输入：

用户问题\
本卦\
变卦\
动爻

输出：

-   卦象含义
-   趋势分析
-   风险提示
-   建议

------------------------------------------------------------------------

## 6 结果页

展示：

-   卦象
-   六爻结构
-   AI解读

操作：

-   分享
-   深度解读
-   再占一次

# 七、六爻算法

生成六爻

    function generateLine() {
      const r = Math.random()
    
      if (r < 0.25) return 6
      if (r < 0.5) return 7
      if (r < 0.75) return 8
      return 9
    }

生成六次

    const lines = []
    for (let i = 0; i < 6; i++) {
     lines.push(generateLine())
    }
